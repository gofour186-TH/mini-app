# FXBridge Telegram Mini App

Статический Telegram Mini App для `FXBridge` в светлом premium-стиле с вайбом `Ex24`.

## Что внутри

- `Экран 1`: главная с карточками, кнопкой `Обменять`, фильтрами стран, списком направлений и нижним меню
- `Экран 2`: создание заявки
- `Экран 3`: чат с менеджером
- `Экран 4`: мои сделки
- HTML/CSS/JS без бэкенда
- моковые данные, готовые для демонстрации
- совместимость со статическим деплоем на Netlify

## Файлы

- `index.html` - вся структура интерфейса
- `styles.css` - мобильный светлый UI для Telegram WebApp
- `app.js` - навигация по экранам, моковые данные и интерактив

## Как задеплоить на Netlify

1. Открой Netlify
2. Нажми `Add new site`
3. Выбери `Deploy manually`
4. Перетащи папку `telegram-mini-app`
5. Получи публичный `https` URL

## Как подключить к Telegram-боту

1. Размести приложение на Netlify
2. Возьми публичную ссылку
3. В `@BotFather` укажи Web App URL для своего бота
4. При необходимости обработай `Telegram.WebApp.sendData()` в боте
